# 🚀 Sales & Profit Analysis Dashboard | Excel Project

## 📌 Project Overview

This is my first end-to-end data analysis project using **Microsoft Excel**.

The project focuses on analyzing sales and profit performance from a business perspective to generate actionable insights that support better decision-making.

The dashboard was built using advanced Excel tools and techniques including:

* Power Query
* Power Pivot
* Data Modeling
* Pivot Tables
* Interactive Visualizations

---

# 🎯 Project Objectives

* Analyze overall sales performance
* Evaluate profitability across regions and categories
* Track Year-over-Year (YOY) growth
* Identify top-performing states and sub-categories
* Build an interactive dashboard for business analysis

# 💡 Key Learnings from the Project

## 📌 1. Strong Data Modeling is Essential

Building a proper data model is the foundation of successful data analysis.

---

## 📌 2. Measures Provide More Flexibility

Using Measures instead of static calculations improves:

* Flexibility
* Accuracy
* Scalability

---

## 📌 3. Profit & Profit Margin Must Be Analyzed Together

Analyzing profit alone is not enough.

Combining:

* Profit
* Profit Margin

provides deeper business insights and helps improve performance.

---

## 📌 4. Dashboard Simplicity Matters

Creating a clear, interactive, and user-friendly dashboard makes it easier to:

* Understand data
* Explore insights
* Support decision-making
